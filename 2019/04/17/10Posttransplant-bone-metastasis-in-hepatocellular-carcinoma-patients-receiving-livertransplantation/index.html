肝癌肝移植术后骨转移患者的手术治疗

Posted on 2019-04-17 | Post modified: 2019-05-15 | In 临床研究

说明：

  1. 参考自李照,高杰,孙馨.肝癌肝移植术后骨转移患者的手术治疗.中华普通外科杂志,2013,28(3):193-195.

一、资料与方法

（一）一般资料

  1. 回顾性分析2001年7月-2010年4月北京大学人民医院肝胆外科和骨肿瘤科的20例肝移植术后肝癌骨转移行手术治疗患者的临床资料，其中男18例，女2例，年龄32〜63岁，平均45.1岁。

  2. 肝移植原发病均为原发性肝癌合并肝炎后肝硬化。

  3. 肝癌患者中：

    • 16例超出米兰标准，合并微血管癌栓。

    • 4例符合米兰标准。

      米兰标准：

      （1）单个肿瘤直径不超过5cm

      （2）或肿瘤数目少于3个且最大直径不超过3cm

      （3）无大血管侵犯、淋巴结或肝外转移

  4. 肝移植术后常规应用三联免疫抑制治疗方案，并于1个月内逐步停用激素。

    • 环孢素或他克莫司
    • 吗替麦考酚酯
    • 激素

  5. 肝癌骨转移病灶均经术前全身骨扫描或FDG PET-CT检查发现，手术切除后的病理结果予以证实。

（二）骨转移病灶的治疗方式

  1. 全部患者均接受了手术治疗，其中：
    • 中轴骨病灶均行肿瘤后路切除内固定术(19例)
    • 四肢骨行肿瘤切除髓内针内固定术(1例)
  2. 有13例患者接受了联合放疗(姑息性分割放疗或适形放疗)。
  3. 5例患者接受全身化疗(5氟尿嘧啶+顺铂+阿霉素)。

（三）疼痛评价

  1. 患者疼痛强度采用数字分级法(NRS)
    • 用0〜10的数字代表不同程度的疼痛，0为无痛，10为最剧烈疼痛
  2. 疼痛缓解评价：
    • 疼痛评分下降5分或以上为显效
    • 疼痛评分下降低于5分为有效
    • 疼痛评分无下降或上升为无效

（四）随访

  1. 随访记录了患者肝移植术后至发现骨转移的时间、从诊断骨转移至患者死亡或最后随访的时间。
  2. 所有患者均得到随访，随访率为100％，中位随访时间为12.1个月。

（五）统计学方法

  1. 应用χ2检验来分析手术治疗效果
  2. 生存分析采用Kaplan-Meier法
  3. 单因素分析应用Log-rank方法
  4. 多因素分析采用COX回归模型

二、结果

（一）肝癌肝移植术后骨转移患者的临床特征

  1. 就诊患者的症状有：
    • 疼痛(15例，75％)
    • 截瘫或大小便失禁(2例，10％)
    • 下肢无力(1例，5％)
    • 病理性骨折(1例，5％)
    • 1例(5％)没有特殊症状
  2. 骨转移病灶的部位：
    • 19例(95％)位于中轴骨
      • 颈椎3例
      • 胸椎11例
      • 腰椎5例
    • 1例(5％)位于四肢骨
      • 股骨
  3. 其他转移灶：
    • 11例(55％)合并肋骨、髂骨多发骨转移
    • 10例(50％)合并其他脏器转移
      • 肝转移 2例
      • 肺转移 6例
      • 脑转移 1例
      • 腹腔内淋巴结转移 1例
  4. 影像学显示骨转移病灶均呈溶骨性破坏。
  5. 患者中术前检查发现：
    • 碱性磷酸酶升高者12例(60％)
    • 血钙增高者4例(5％)

（二）肝癌肝移植术后骨转移患者手术治疗的生存率

20例肝癌肝移植术后骨转移的患者手术治疗后的生存时间为1〜30个月，中位生存时间7.5个月，1年累积生存率仅为20％，见图1。

（三）肝癌肝移植术后骨转移患者的手术疗效

  1. 手术治疗后疼痛缓解：
    • 显效者10例(67％)
    • 有效者4例(26.6％)
    • 无效者1例(6.7％)
  2. 患者行骨转移病灶术后恢复顺利，无明显手术相关并发症。
  3. 同时期未行手术治疗肝移植术后骨转移患者18例
    患者(化疗或放疗)中有：

    • 10例疼痛缓解有效

    • 8例疼痛缓解无效

      两组之间比较差异有统计学意义(χ2=5.887，P<0.05)。

（四）肝癌肝移植术后骨转移患者预后的单因素和多因素分析结果

  1. 单因素相关分析
    • 患者的肝移植时肝癌是否合并微血管浸润
    • 骨转移时的MELD评分(model end-stage liver disease，MELD)(终末期肝病模型)
    • 发生骨转移的时间
    • 是否合并其他脏器转移
    • 是否合并放化疗
  2. 单因素相关分析结果，下列因素可能影响患者预后的因素(分别F=4.724,3.435，均P<0.05)。
    • 骨转移时的MELD评分
    • 是否合并微血管浸润
  3. COX回归分析，结果只发现是否合并微血管浸润是影响这类患者的预后因素(β=2.589，P<0.05)。

（五）转归

  1. 所有患者全部随访，至随访截止时20例患者全部死亡。
  2. 死亡原因为：
    • 肝功能衰竭15例(75％)
    • 肺部感染合并呼吸功能衰竭4例(20％)
    • 脑疝1例(5％)

三、讨论

（一）肝移植术后肝癌骨转移的一般特点

  1. 文献报道肝癌骨转移的发生率在8％〜16％，而肝移植术后的患者处于免疫抑制状态，骨转移发生率明显较高。
  2. 肝癌骨转移的首见症状为疼痛，本组病例中有75％的患者因疼痛就诊发现。
  3. 骨转移病灶最常见的部位椎体、肋骨、骨盆、长骨等：
    • 本组中有95％患者骨转移病灶位于中轴骨。
    • 其中有2例患者累及脊柱较重甚至合并截瘫。
  4. 一般认为血清碱性磷酸酶对骨转移诊断有参考价值，本组所有患者影像学均观察到溶骨性破坏表现，同时有60％的患者出现碱性磷酸酶增高，表明碱性磷酸酶在此类患者的早期诊断中有重要作用。

（二）肝移植术后肝癌骨转移的治疗

  1. 对于肝癌骨转移的常见治疗方法包括：

    • 双膦酸盐药物治疗

    • 放疗

    • 外科手术

      均属于姑息对症治疗

  2. 对于中轴骨转移合并截瘫风险的这类患者药物治疗和放疗效果往往不佳，实施骨转移病灶手术治疗能够达到改善生活质量的目的，同时对一般状况好、单发骨转移患者手术也能够明显提高患者的预后。
  3. 本组研究中通过实施骨转移病灶切除内固定术
    • 有14例患者疼痛症状得到明显改善
    • 其中1例截瘫患者恢复肢体感觉和运动
    • 仅有1例截瘫患者因为肿瘤压迫时间太长而不能恢复
    • 与非手术治疗相比具有显著的手术疗效
    • 这表明对这类特定的患者选择手术治疗能够明显提高患者的生活质量

（三）肝移植术后肝癌骨转移的预后及影响因素

  1. 总体来说，一旦发生肝癌骨转移，患者的预后均较差。
    • 国内学者报道肝癌肝移植术后1、2、3年存活率分别为91.2％、68.5％、59.4％
    • 本组肝移植术后骨转移患者1年存活率仅为20％，明显低于肝癌肝移植患者的平均水平。
  2. 通过对可能影响这类患者的预后因素进行单因素分析发现，发生骨转移时患者的MELD评分>25分的患者预后较差。
  3. 通过对患者的死亡原因分析我们也可以发现，患者的主要死亡原因为肝功能衰竭(本组占75％)或肝外转移所致的其他脏器功能衰竭，因此我们认为患者骨转移时的肝功能状态影响了患者的预后。
  4. 米兰标准是目前世界范围内肝癌肝移植的公认标准，符合米兰标准的肝癌肝移植患者5年生存率在75％以上，肝癌复发率<10％，由于米兰标准没有考虑肿瘤是否合并血管浸润及肿瘤的生物学特征，目前对于其还存在争论。
  5. 我们早期对肝癌肝移植的指征条件过宽，本组有16例超过米兰标准，很多合并血管癌栓的患者也选择了肝移植手术，导致这类患者肿瘤早期复发，肝内广泛转移后导致肝功能迅速恶化，因此骨转移手术后患者预后也较差。
  6. 目前对于肝癌骨转移的放射治疗，其原理一般为：
    • 促进胶原蛋白合成
    • 继之产生大量血管纤维基质
    • 成骨细胞活性增加而形成新骨
  7. 但肝癌骨转移灶的外照射是姑息性的减症治疗，因此在治疗方案上要多方面权衡考虑，以既能止痛、又不出现放疗不良反应为目的，尤其是椎体、骨盆区的放疗，需要兼顾重要脏器的保护，因此也有研究认为放疗的主要作用
    是缓解或减轻局部疼痛，提高患者生存质量，但不能有效延长生存时问，这方面与我们的结果相符。

（四）总结和局限性

  1. 我们认为对于没有微血管浸润、肝功能良好、累及中轴骨严重影响生活质量的肝移植术后肝癌骨转移患者可以选择积极的手术治疗，能够提高这类患者的生活质量。

  2. 鉴于本回顾性研究只是针对肝移植术后肝癌骨转移的手术患者，入组病例数少，因此需要还进一步增加样本量并设计对照研究以得出更合理的结论。

四、附：终末期肝病模型（MELD）评分

  1. 肝硬化根据是否发生腹水、肝性脑病及食管胃底静脉曲张破裂出血分为代偿期和失代偿期肝硬化。终末期肝病多指失代偿期肝硬化。
  2. 终末期肝病的表现包括：
    • 黄疸
    • 腹壁静脉曲张
    • 皮肤色素沉着
    • 下肢水肿
    • 腹水
    • 蜘蛛痣
    • 杵状指
    • 肝掌
  3. 利用Child-Pugh分级虽然能够判断肝病患者肝脏的代偿能力，但对肝病严重程度的判断仍然不太精确。
  4. 自2000年美国Malinchoc等最初创立终末期肝病模型（Model for end-stage liver disease ，MELD）评分以来，随后的研究证实其为不同的终末期肝病生存率的预测指标。
    • 其可对终末期肝病短期、中期死亡率进行有效的预测
    • 其评价指标获得简单、客观、易于计算，在肝病诊治中广为应用
  5. 终末期肝病模型（MELD）是以肌酐、国际标准化比值（INR）、胆红素结合肝硬化病因来评价慢性肝病患者肝功能储备及预后的评分系统。
  6. 终末期肝病模型（MELD）评分的计算公式为：MELD=3.78×ln [T-BiL(mg/dl)]+11.2×ln[INR]+9.57×ln[Cr(mg/dl)]+6.43（病因：胆汁淤积性肝病或酒精性肝病为0，病毒等其他原因肝病为1）。
    • 公式中的T-BiL为总胆红素，INR为国际标准化比值，Cr为血清肌酐，ln为自然对数。
    • 需要注意的是，在欧美国家，公式中总胆红素和血清肌酐的单位用的是“mg/dl”。而我国使用的是国际单位制(SI)，总胆红素和血清肌酐的单位是“μmol/L”。在使用该公式计算MELD分数时，需要先把总胆红素和血清肌酐“μmol/L”换算为“mg/dl”。总胆红素的换算系数为17.1，血清肌酐的换算系数为88.4。
    • 因此，在使用国际单位制时，MELD计算公式为：
      • MELD=3.78×ln[T-BiL(μmol/L)÷17.1]+11.2×ln[INR]+9.57×ln[Cr (μmol/L)÷88.4]+6.43（病因：胆汁淤积性肝病或酒精性肝病为0，病毒等其他原因肝病为1）。
    • MELD评分越高，其风险越大，生存率越低。
